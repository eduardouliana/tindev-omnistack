# tindev-omnistack

# Módulos para serem instalados
* yarn init -y
* yarn add express
* yarn add nodemon -D
* yarn add mongoose // Conexão com mongoDB
* yarn add axios // Requisições rest
* yarn add cors // Permitir que seja acessada de qualquer endereço
